# Word Coding Game (c)

This is a logic based word Coding Game built using the C programming language. The game challenges users to identify patterns between words and numbers and decode new words based on the given logic.

##Features
. Easy, Medium, and Hard levels
. Score tracking system
. Pattern recognition- based questions.
. Interactive command- line interface.

## How it Works
The game assigns numbers to letters based on patterns.

Example:
CAT = 123
DOG = 456

Now the user has to find:
COT = 153

The Player must analyze the pattern and enter the correct answer.
